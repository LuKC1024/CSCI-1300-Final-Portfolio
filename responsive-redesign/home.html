Redesign the Popeyes Website

I found the Popeyes website has some usability problems so redesign it. This page describes the redesigning process.

Identify Usability Problems

Prototype

To redesign the Popeyes website, I start by creating a low-fidelity prototype. This grayscale and simplistic prototype focuses on the overall structure of web content and summarizes solutions to the problems mentioned in the last section. Later I refine this prototype to create a high-fidelity prototype, which should give all details that should be presented in the final HTML implementation. I could have gone directly to HTML from the lo-fi prototype. But Figma is an amazing tool that speeds up the refinement of prototypes and exploring alternative decisions on details.
